Creativerse Sprites Picker

Last updated February the 20th, 2018 - all of the following sprites should work correctly in the game!

As far as I can tell, some sprites are currently missing (they don't appear here and they don't work in the game).

This tool lets you pick up sprite codes to be put into Arc Signs in the game Creativerse, free to play on Steam.

Check out the CreatiSign Generator to import your images into Arc Signs!

Click on any icon to have its code appended to the yellow textarea below, then copy those codes into the Arc Signs and alter them as you please.

If you want to space sprites so that each goes on a different block you can use this code before them:

<size=8><width=8><offset=1.5><zoffset=0.5><mspace=1><line-height=1>

fiddle with the "size", "offset" and "zoffset" values, the other tags should be left alone unless you know what they do and how they work.
